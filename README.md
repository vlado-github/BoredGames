# BoredGames
BoredGames platform for creating custom board games.

## BoredGames.Server
Backend server for handling queueing, game state and players actions. 

## BoredGames.Client
### Build BoredGames.Portal
1. Run shell or cmd
2. Position to source folder of project
3. Execute command:
`yarn build`

### Run BoredGame.Portal
1. Run shell or cmd
2. Position to source folder of project
3. Execute command:
`yarn dev`
